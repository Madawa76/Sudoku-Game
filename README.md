🧩 Python Sudoku Game (GUI – Advanced)
A fully featured Python Sudoku game built with Tkinter, offering multiple difficulty levels, smart hints, auto‑solve, themes, and save/load functionality.
This project is ideal for learning Python GUI development, game logic, and structured application design.

📌 Features
✅ Core Gameplay

Classic 9×9 Sudoku board
Proper 3×3 sub‑grid separation
Numbers validated (only 1–9 allowed)
Playable using keyboard or mouse

✅ Difficulty Levels

Beginner – More cells filled
Intermediate
Expert – Fewer clues, higher challenge

✅ Advanced Enhancements

⏱️ Game Timer
💡 Hint Button (reveals one correct cell)
✅ Check Solution (highlights incorrect cells)
🤖 Auto‑Solve (backtracking-based)
💾 Save & Load Game Progress
🌗 Dark / Light Theme Toggle
🔁 Reset & main menu navigation


🛠️ Technologies Used

Python 3.x
Tkinter (built‑in Python GUI library)
Standard libraries only:

random
copy
json
time



✅ No external dependencies required.


🎮 How to Play

Select a difficulty level from the menu
Click a cell and enter numbers 1–9
Use buttons to:

Hint → Reveal one correct value
Check → Highlight wrong answers
Solve → Automatically complete the puzzle
Save / Load → Resume later
Theme → Switch Dark / Light mode
